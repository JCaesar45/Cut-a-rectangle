Algorithm Details

This algorithm counts the number of ways to cut an m×n rectangle into two congruent pieces that are related by 180° rotation. The solution uses dynamic programming with state machines to efficiently count valid dissections.

Complexity Analysis

Time: O(m×n×2^n)
Space: O(2^n)
